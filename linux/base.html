    # linux 基础

    # 文件管理

    * 目录结构
    * bin 可执行命令
    * boot 核心文件 开机文件
    * dev 设备
    * etc 配置
    * home 其他用户（除root）
    * lib 调用库
    * root root用户文件
    * sbin root用户的执行文件
    * srv 
    * temp 临时存放
    * opt 第三方软件安装目录
    * media 存储 u盘
* 命令
    * pwd
    * cd ./当前目录 ../上层目录 - 上一个工作目录 ~当用户
    * mkdir 创建目录
        * mkdir -p 嵌套创建 mkdir -p /temp/a/b
    * rmdir 删除目录 只能删除空目录
    * ls 文件名|目录名|没有
        * -a 所有文件 包括隐藏文件
        * -l 长格式 详细信息
        * -i inode number
    * chmod
    * chown 
    
    * cp 拷贝文件,可以同时重命名 空格 隔开 多个文件 cp from to 
        * cp -r 拷贝目录
        * cp -p 完全拷贝 包括权限
    * mv 移动文件 移动目录不需要 -r
    * rm 删除文件
        * rm -R 删除目录包含文件
        * rm -fR 强制删除
    * touch 创建空文件 如果文件已存在则修改时间戳
* 文件链接
    * ln -s [原文件][链接名]符号链接 相当于快捷方式
    * ln  [原文件][链接名] 硬链接
* 文件压缩与解压
    * tar -cf[打包文件名] [要打包的文件空格分开] 不会亚索文件
          -tf 列出打包文件
          -f --delete 删除文件
          -f -A 合并
          -f -r 添加
          -xf 解压文件
          -xf  -C 指定目录
    * gzip 压缩文件 不能压缩目录
        * gzip -9 压缩比
    * gzip -d 解压文件
    
    * tar -czf （z）打包并压缩文件
    * tar -xzf 解压文件
* 查找文件
    * locate [路径][文件名关键字]
        * updatedb 更新 数据库中存储的文件名
        
    * find [范围(目录)][条件][动作]
        * find / -name password  精确
        * find /bin -type l (d目录 f普通文件 c字符设备 b块文件 l符号链接)
        * find /tmp -mtime -3 -ls 那天之内 mmin 分钟
        * find /tmp -mtime +7 -ls n天之前
        * find /tmp -mtime 4 -ls  第n天
        
        * find /tmp -size +3k -ls
        * find /tmp -size -3k -ls
        * find /tmp -size +3k -ls
        
        * find /tmp -user root
        * find /tmp -uid 500
        * find /tmp -group admin
        * find /tmp -gid 0
    * 通配符
        * ”*“ 任意长度字符串
        * "?" 单个字符
        * [c1-c2] c1-c2 中的任意单一字符 [a,b,c] 列表中的任意字符
        * {str1，str2，str3} 任意一个字符
        * ! 取反
    * 查看文件内容
        * cat
            * 连接文件
            * 从键盘创建文件
            * cat file 查看文件 一次显示全部
        * less q退出
        * head -n 20 file 默认10行
        * tail 
        
    * vi
        * vim 
            * 一般模式
            * 输入模式 输入
            * 命令行模式
        * 移动光标 hjkl 左下上右
        * x 删除 i 进入输入模式 dd 删除整行 J 删除换行 u 撤销最后一次编辑 ctrl+r 取消撤销 :wq 保存退出 :q!退出不保存
        * Set 环境参数
        * Set no 环境参数 取消环境参数
        * gg 文件首行 G 文件末行 0 当前行首 $ 当前行尾 3G第三行行首
        */ 进入命令行 输入man 查找包含man的内容 n下一个 N 上一个
        * nyy 复制光标开始向下n行
        * p 在光标的下一行黏贴
        * dd 删除行 ndd 删除光标开始的向下n行
        * x 删除光标所在处字符 nx 光标开始删除n个
        * u 撤销一次操作
        * 命令行下 e! 撤销所有修改
        * 命令行下 w 新文件名 
    * 重定向 1> filename 1标准输出 重定向到filename 覆盖 >> 添加到文件末尾 2> 错误输出 &> 同时输出标准和错误
    * /dev/null 系统黑洞 输出到null文件都会被丢弃
    * 输入重定向 < 和  <<  
        * tr 替换
        * sort -k 排序
    * 管道 特殊的输入输出重定向
        * 管道操作符 |
        * grep [关键字][目标文件]文本查找命令 ls -al /etc |grep 'ssh'
        * cut 文本切割 cat install.log |cut -c1,2 前两列
        * wc 文本统计
    * 用户
        * id 用户名 户名信息
        * finger 用户名
* 查看硬盘
    * ls -l /dev/xxy xxy硬盘名
    * ls -l /dev/xxyn 分区
    * fdisk -l /dev/sda
* 硬盘分区
    * fdisk /dev/sdb
        * n 新建分区 p 主分区 e 扩展分区 l 逻辑分区 w 写入分区表 1
        退出 l 列出支持分区 t 修改分区类型
    * 创建文件系统
        * mkfs -t ext4 /dev/sdb1 格式化成ext4文件系统
        * mkswap /dev/sdb7 交换分区
    * 挂载分区
        * mount -t ext4 /dev/sdb1 /myhardisk/books
        * -o 选项 设置挂载属性
    * 卸载分区
        * umount /dev/sdb8
    * 开机自动挂载
        * /etc/fstab 文件 
            * 字段1 要挂载的设备
            * 字段2 挂载点
            * 字段3 文件系统类型
            * 字段4 挂在参数
            * 字段5 dump
            * 字段6 开机检查顺序 0 跳过
        * /dev/sdb1 /project/finished ext4 default 0 0 
            *  标签 与 uuid
                * e2label 未分区添加标签
    * 网络接口
        * ifconfig -a
        * infconfig eth1 up 开启接口 down 关闭
        * ifconfig  eth1 ip netmask ip
    * 软件安装
        * rpm -q opera 查询
        * rpm -ivh  v 显示安装信息 h #号进度条
        * rpm -e opera（全名）卸载
        
        * yum clean all
        * yum makecache
        * yum install
        * yum update
        * yum remove
    * 查看进程
        * ps -Af 列出所有进程
        * kill pid
        * ps -Af|grep sshd
        
        * top 动态监视进程

    apt-get update

sudo  update-rc.d test defaults 95
